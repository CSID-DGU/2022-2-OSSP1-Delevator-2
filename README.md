## 프로그램 기능
- 얼굴인식 및 객체인지 모델을 통해 핸드폰, 교안 등 부정행위에 해당되는 물체 인식

- 여러 응시자에 대해 감독자는 각각의 링크를 통해 감독할 수 있고 부정행위 알림을 받을 수 있음
